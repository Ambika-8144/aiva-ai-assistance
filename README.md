# 📖 Overview

AIVA (AI Virtual Assistant) is an intelligent assistant designed to provide AI-powered interactions, automation, and smart assistance through modern Large Language Models (LLMs).

The project combines powerful AI capabilities with a user-friendly interface to create an interactive assistant capable of understanding user queries and generating intelligent responses.

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/aiva-ai-assistant.git
cd aiva-ai-assistant
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Environment

Windows:

```bash
.venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python webui.py
```

or

```bash
streamlit run webui.py
```
